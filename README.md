# javascript-Calculator
A sinple Javascript Calcualtor that runs as a bookmarklet in the browser

Capable of most scientific functions. (Not currently in the UI)
But can handle trig (currently only as radians) and exponets. (log for log base 10, ln for log base e)
has apriximatly 10 digits of accuracy. (1e2 = 100, 1e-2 = 0.01)
Can use sqrt(1) or √1, pi, e, and ans (previous answer)

To install, simply create a new bookmark and copy the source code into the url section, then save and click the bookmark.
